启动docke-compose:

docker-compose ps        # 查看状态
docker-compose logs -f   # 实时查看日志

三者关系：

docker-compose    ← 你写配置，管多个容器
    ↓
docker client     ← 发指令
    ↓
docker daemon     ← 真正干活的引擎
